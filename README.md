# ✅ TodoTS

## 🌟 Features | Características

- ✨ Añadir, editar y eliminar tareas
- ✅ Marcar tareas como completadas
- 🔍 Filtrar por: Todas/Activas/Completadas
- 🧹 Limpiar tareas completadas
- 💾 Persistencia simulada
- 🎬 Animaciones fluidas
- 📱 Totalmente responsive
